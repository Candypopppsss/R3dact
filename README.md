##R3DACT
R3dact: AI-Powered Cybersecurity Platform

Architecture: Built a multi-layered security agent using React, Express, and TypeScript, featuring a modular engine system for threat detection and behavior reasoning.
Detection Logic: Developed robust heuristic algorithms to identify typosquatting, URL obfuscation, and social engineering indicators with an emphasis on explainable AI (XAI).
Data Persistence: Leveraged SQL.js to create a lightweight, in-app database for real-time attack history tracking and statistical analysis.
UX/UI: Created a professional dashboard using modern CSS and React hooks, providing a seamless interface for threat analysis and situational awareness.
